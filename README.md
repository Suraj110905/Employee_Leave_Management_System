# Employee Leave Management System

A web-based HR platform that allows employees to apply for leave, managers to approve or reject requests, and HR administrators to manage leave policies, holidays, leave balances, and reports.

## Project Objective

The objective of this project is to automate the employee leave management process and reduce manual work, approval delays, and errors in leave balance tracking.

## Main Users

- Employee
- Manager
- HR Admin

## Core Features

### Employee
- Login
- Apply for leave
- View leave balance
- View leave status
- View leave history

### Manager
- View leave requests
- Approve or reject leave
- Add comments
- View team absence calendar

### HR Admin
- Manage employees
- Manage leave types
- Manage holidays
- View reports
- Configure leave policies

## Tech Stack
